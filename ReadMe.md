Steps on how to create a Github Pages:
1. Join the classroom invitation to have remote repository.
2. Clone your repository to your local machine.
3. Create html and css file in your local repository.
4. Add, commit and push to remote repository,
5. Go to settings of your repository and make it public(can be found at the bottom most of the settings).
6. Go to Pages part in the navigation bar in the settings and change the branch to main where the html and css file is located.
7. The link will be provided for your site.

Key takeaways:
    Flexbox is helpful in layouting the website as it proposes many reliable functions.
